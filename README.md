# #저장경로
![image](https://github.com/leegowoon/scss/assets/145514701/de51fb12-c227-455b-a937-0ae25dee5817)

# #"savePath": null, --> null이면 scss파일과 같은 위치에 style.css가 생긴다.

# #"savePath": "~/css", --> style.scss를 의미, /는 style.scss있는 폴더를 의미,  scss파일과 같은 위치에 css폴더가 생성되고 그 css폴더안에 컴파일된 style.css가 생긴다.
![image](https://github.com/leegowoon/scss/assets/145514701/7ace5815-9249-462d-ad18-226c9be821b0)

![image](https://github.com/leegowoon/scss/assets/145514701/50b16455-2bfe-466d-9868-ba8b38a5f57c)

# #"savePath": "~/../css" --> style.scss있는 폴더(부모)와 같은 위치에 있는 css폴더가 생성되고 그 css폴더안에 컴파일된 style.css가 생긴다.
